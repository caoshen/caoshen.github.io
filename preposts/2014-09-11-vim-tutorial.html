lesson 1

编辑

h j k l
左 下 上 右

i / I 插入模式
i 光标处插入文字
I 行首插入

x / X 删除
X 删除光标左边字符

a / A 追加
A 在行尾追加

u 撤销
ctrl + r 恢复

0 / $ 跳到行首/尾

^ 跳到本行第一个分空字符

w / b / e
下一个 word / 前一个 word / word 尾

W / B / E
下一个 WORD / 前一个 WORD / WORD 尾

word 指连续字母或者连续标点符号，比如 abc , :: 是 word
WORD 指连续非空字符， 比如 foo(bar) , xyz__t 是 WORD

r / R 替换
r 替换单个字符
R 替换多个字符

:w / :q 保存 / 退出

lesson 2

操作和重复

f / F
使用方法：
fx 在当前行,光标向前（右）寻找 x 第一次出现的地方。
Fx 在当前行,光标向后（左）寻找 x 第一次出现的地方。

t / T
类似 f / F，不过找到的是第一次出现的地方的左 / 右边 (till)。

d 删除
一般和表示位置的动作连用
dw 删除一个词
dd 删除一行
D 删除到行尾
df- 删除到 - 第一次出现的位置（包括 -）

c 删除后进入插入模式

. 在当前位置，重复上一次的编辑命令

lesson 3

复制和粘贴

y 复制
Y 复制一行

p 粘贴
p / P 光标之后（前）粘贴

y 可以在 可视模式 中使用

d c x 在删除的同时也复制了内容

“ 和 a - z 加上 y
选择一个寄存器来复制

A - Z 加上 y
追加复制 (append-copy)

“* 或 “+ 选择系统剪切板

o / O 在当前行下方（上方）插入

lesson 4

搜索

/ 搜索

? 向后搜

n / N 跳到下（上）一个搜索位置

  • / # 向前（后）搜索 光标所指的表达式

搜索的是正则表达式
如
a*b 表示 0 个或多个 a 后面跟一个 b
^abc 表示 ‘abc’ 出现在行首
[0-9] 表示 0 - 9 中的一个数字

lesson 5

标记和宏

m 后面跟 a - z 可以设置标记

` 跳到标记

‘ 跳到标记所在行的行首

q 后面跟 a - z 可以记录
再次使用 q 停止记录

@ 加 a - z
使用记录的宏

lesson 6

其他动作

% 在括号之间跳转 () , [] , {}

H M L 跳到屏幕 上方 中间 下方

G 跳到文件尾，或第 # 行
比如
10G 跳到第 10 行

gg 跳到首行

  • / + 跳到上 / 下一行

K 跳到光标所指表达式的帮助文档（如果有）

( 和 ) 跳到上（下）一个句子

{ 和 } 跳到上（下）一个段落（空行）

句子以 . ! ? 结尾

段落以 空行 结尾

[[ 跳到上一个处在第 0 列的 {
]] 跳到下一个处在第 0 列的 {

lesson 7

其他命令

J 连接当前行和下一行
或者 连接可视模式块中的所有行

r / R 替换一个 / 多个字符

C 等于 c$
改变到行尾（删除后插入）

D 等于 d$
删除到行尾

Y 等于 yy
复制一行

s / S 删除一个字符 / 一行 后进入插入模式

/ < 缩进（取消缩进）

= 自动格式化

~ 改变大小写
