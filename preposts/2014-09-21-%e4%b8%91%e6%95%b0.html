丑数

发表于 2014-09-21   |   分类于 ， ，   |  

题目描述：

把只包含因子2、3和5的数称作丑数（Ugly Number）。例如6、8都是丑数，但14不是，因为它包含因子7。
习惯上我们把1当做是第一个丑数。求按从小到大的顺序的第N个丑数。

输入：

输入包括一个整数N(1<=N<=1500)。

输出：

可能有多组测试数据，对于每组数据，
输出第N个丑数。

样例输入：

3

样例输出：

3

p2 p3 p5 分别保留 2 3 *5 后第一个大于 a[n - 1] 的位置

例如

当前 a = [1, 2, 3, 5]，则 p2 = 2, p3 = 1, p5 = 1
