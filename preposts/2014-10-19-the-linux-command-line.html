The Linux Command Line

TLCL

一些简单命令

date 显示时间

星期 + 月 + 日 + 时：分：秒 + 时区 + 年

cal 显示日历

df 查看磁盘剩余空间的数量

free 显示空闲内存的数量

exit 结束终端会话

文件系统中跳转

pwd 打印出当前工作目录名

cd 更改目录

cd 更改工作目录到主目录

cd - 更改工作目录到先前的工作目录

cd ~user_name 更改工作目录到用户主目录

ls 列出目录内容

操作系统

ls dir 列出指定目录下的文件

ls dir1 dir2 dir3 列出多个目录下的文件

ls -l 结果以长模式输出，给出更多细节 （权限，所有者，大小，修改时间等）

多数命令的形式像这样：

command -options arguments

也支持长选项，长选项由两个中划线和一个字组成，比如

ls -lt –reverse

t 选项按文件修改时间的先后顺序

reverse 会以相反的顺序输出

ls -a 显示所有文件，包括隐藏文件

ls -d 显示当前目录的信息，通常和 -l 连用，可以看到目录的详细信息

ls -F 给每个所列出的名字分类，添加一个指示符

ls -h 和 -l 连用，以可读的格式来显示文件大小

ls -r 以相反顺序来显示结果，通常，ls 命令的输出结果按字母升序排列

ls -S 按文件大小排列 (size)

ls -t 按修改的时间顺序排列

file 打印文件内容的简单描述

less 浏览文件，可以上下翻页

page up or b 向后翻滚一页 backward

page down or space or f 向前翻滚一页 forward

up arrow 向前移动一行

down arrow 向后移动一行

G 移动到最后1行

1G or g 移动到第一行

/characters 查找字符串

n 向前查找下一个出现的字符串

h 帮助

q 退出 less 程序

less is more. less 程序是 more 程序的加强版

linux 系统中的目录

/ 根目录

/bin 包含系统启动和运行所必须的二进制程序

/boot 包含 linux 内核，最初的 RMA 磁盘映像 （系统启动时，由驱动程序所需），和启动加载程序。

/dev 包含设备节点的特殊目录。“一切都是文件”，也适用于设备。在这个目录里，内核维护着它支持的设备

/etc 包含所有系统层面的配置文件。也包含一系列的 shell 脚本，在系统启动时，这些脚本会运行每个系统服务。

/home 用户目录

/lib 包含核心系统程序所需的库文件。这些文件与 windows 中的动态链接库相似

/lost + found 每个使用 linux 文件系统的格式化分区或者设备，例如 ext3 文件系统，都会有这个目录。当恢复一个损坏的文件系统时，会用到这个目录。

/media 包含可移除媒体设备的挂载点，如 usb， cd-rom

/opt 用来安装可选的软件

/proc 由 linux 内核维护的虚拟文件系统

/root root 账户的主目录

/sbin 包含系统的二进制文件。通常为超级用户保留

/tmp 存储各种程序创建的临时文件

/usr 包含普通用户所需要的所有程序和文件

/usr/bin 包含系统安装的可执行程序

/usr/lib 包含 /usr/bin 中的程序所使用的共享库

/usr/local 非系统发行版自带，却打算让系统使用的程序的安装目录。 通常，由源码编译的程序会安装在/usr/local/bin 目录下。

/usr/sbin 包含许多系统管理程序

/usr/share /usr/bin 中程序使用的共享数据

/var 可能需要改动的文件存储的地方。各种数据库，假脱机文件，用户邮件等

/var/log 包含日志文件，各种系统活动的记录
