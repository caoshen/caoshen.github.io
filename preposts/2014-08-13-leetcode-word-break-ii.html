Word Break II

发表于 2014-08-13   |   分类于 ， ， ，   |  

Given a string s and a dictionary of words dict, add spaces in s to construct a sentence where each word is a valid dictionary word.

Return all such possible sentences.

For example, given
s = “catsanddog”,
dict = [“cat”, “cats”, “and”, “sand”, “dog”].

A solution is [“cats and dog”, “cat sand dog”].

prev 记录 s[: i] 是否可分为 s[:j] , s[j:i]。
当 i = 1..n 时，prev[i][j] 不能确保 s[i:] 可分，需要再次检验后半段；
当 i = n..1 时，prev[i][j] 说明整段都是可分的

以 s = ‘aaaaaaaaaaaab’ d = [‘a’, ‘aa’, ‘aaa’, ‘aaaa’] 为例，
prev[n][0..n-1] = False，不满足下一次递归的条件，直接返回 []
而 prev[1..n][0] 只有 prev[n][0] = False, 其他 prev[1..n-1][0] = True，继续下一层递归
