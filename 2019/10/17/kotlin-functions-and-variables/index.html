可变变量和不可变变量

val 声明的是不可变变量。（value）

var 声明的可变变量。（variable）

val 类似 Java 的 final 变量。var 类似 Java 的普通非 final 变量。

需要注意的是，尽管 val 引用自身是不可变的，但是它指向的对象可能是变化的。类似于 Java 的 final。
