# GitHub Pages 强制 HTTPS

1. 打开 https://github.com/yourname/yourproject/settings, yourname 是 github 账号名，yourproject 是 github 项目名称。
2. 找到 options -> GitHub Pages, 勾选 Enforce HTTPS.
