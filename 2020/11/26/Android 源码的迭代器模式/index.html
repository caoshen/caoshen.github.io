Android 源码的迭代器模式

迭代器模式的介绍

迭代器模式（Iterator Pattern） 又称为游标（Cursor）模式，是行为型设计模式之一。迭代器模式是一个比较古老的设计模式，其源于对容器的访问，比如 Java 中的 List、Map、数组等。

迭代器模式的定义

提供一个方法顺序访问一个容器对象中的各个元素，而又不需要暴露该对象的内部表示。

Android 源码中的迭代器模式
